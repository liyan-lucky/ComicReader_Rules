# 贡献指南

感谢贡献规则、脚本和文档。

## 基本要求

提交前请确认：

1. 内容由你原创，或你有权提交；
2. 不提交漫画图片、章节正文、字体、站点 Logo、APK、账号信息、密钥或其他不应进入仓库的文件；
3. 规则只面向普通公开网页；
4. 不提交用于规避登录、付费、验证码、DRM、加密接口、反爬机制或专有客户端协议的内容；
5. 如参考第三方规则，请写明来源、作者和许可证。

## 允许提交

- 普通公开网页的 CSS、XPath、JSON 路径等读取规则；
- 规则生成脚本；
- 兼容性修复；
- 文档和合规说明。

## 不接受提交

- 漫画内容本身；
- 付费内容复制件；
- 破解、绕过、伪造客户端或规避访问控制的代码；
- 未经授权复制的第三方规则；
- 与本项目无关的站点资产。

## Pull Request

PR 应说明变更目的、影响范围、测试方式和是否引用第三方来源。
