# 第三方来源说明

本文件用于记录仓库中可能引用、参考或自动发现的第三方来源。

## 当前状态

截至本文件创建时，仓库没有声明内置第三方代码库或打包第三方资源文件。

规则文件如由脚本根据公开网页结构自动生成，应仅包含读取规则、域名、选择器、路径或元数据，不应包含漫画图片、章节正文、站点 Logo、字体、APK、账号信息或密钥。

## 贡献者要求

如提交内容参考了第三方规则、脚本、文档或模板，请在 PR 说明和本文件中补充：

- 来源名称；
- 原作者或项目；
- 原始链接；
- 许可证或授权说明；
- 修改内容说明。
