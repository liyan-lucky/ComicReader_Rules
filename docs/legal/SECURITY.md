# 安全策略

## 支持范围

本仓库主要包含公开网页读取规则、索引文件和规则生成脚本。

## 安全问题反馈

如发现以下问题，请通过 GitHub Issue 或仓库维护者公开联系方式反馈：

- 规则或脚本意外包含账号、密钥、Token、Cookie 等敏感信息；
- 自动生成文件包含不应公开的站点资源或用户数据；
- 脚本存在命令注入、路径穿越、供应链污染等风险；
- GitHub Actions 存在权限过高、凭据泄露或危险写入行为。

## 处理原则

维护者会尽快复核并修复。确认存在风险时，可以直接删除文件、回滚提交、吊销相关凭据或临时停止自动生成流程。

## 不属于安全漏洞的内容

普通规则失效、站点结构变更、页面无法读取等问题，请按普通 Issue 提交。
