## 5. 关键发现与建议

### 发现
1. **规则重复严重** — bzmanga.com单域名生成50条完全相同的规则，说明规则生成逻辑有bug
2. **日本站混入** — validate用"漫画"匹配时，日本站页面也有"漫画"（日文汉字同形），title_match已加日文特征词但仍有漏网
3. **搜索URL缺失** — 35条规则中仅1条有searchUrl，无法进行分类搜索扩展catalog
4. **域名-规则比例低** — 24个域名仅35条规则，平均每域名1.5条

### 建议下一步
1. **修复规则生成bug** — 同域名不应生成重复规则
2. **增加搜索URL** — 为更多站点配置searchUrl模板
3. **多轮域名发现** — 用新发现的域名作为种子，迭代发现更多域名
4. **关键词发现** — 从腾讯动漫/快看等排行榜提取更多漫画名作为搜索词
5. **在线catalog扩展** — 用generate_catalog.py的在线抓取模式从各站排行榜/分类页获取更多漫画
